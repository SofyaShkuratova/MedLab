1. Установить PHP и MySQL на компьютере через Open Server:

2. Скачайть и установить Open Server:
    - Перейдите на официальный сайт [Open Server](https://ospanel.io/) и скачайте установщик.
    - Запустите установку и следуйте инструкциям.
    - После установки, запустите Open Server.

3. Настроить PHP:
    - В окне Open Server выберите вкладку "Модули" и установите PHP.
    - Нажмите на кнопку "Настроить" рядом с PHP и выберите нужную версию PHP (например, PHP 7.4).
    - Убедитесь, что включены необходимые расширения (например, php_mysql для работы с MySQL).

4. Настроить MySQL:
    - В той же вкладке "Модули" установите MySQL.
    - Нажмите на кнопку "Настроить" рядом с MySQL и укажите пароль для пользователя root.
    - Запустите MySQL-сервер.
	– Создать базу данных с названием "medlab".
	– Нажать "Импорт", выбрать файл в папке Medlab "medlab.sql".
	- Нажать "Далее". База данных настроена.

5. Настроить папку с проектом:
	– В окне Open Server перейдите на вкладку "Домены".
    - Скопировать папку "Medlab" (папка с php файлами) и вставить ее в папку C:\OpenServer\domains.
	– Открыть браузер и перейти по адресу локального домена ( http://medlab/home.php).
    - Вы должны увидеть результат выполнения вашего PHP-кода.
